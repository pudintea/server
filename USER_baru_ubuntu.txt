
Membuat User Baru
===================
> sudo adduser nama_user

Ganti nama_user dengan nama user yang ingin kamu buat. Perintah ini akan:
- Membuat user baru
- Membuat home directory (misalnya /home/nama_user)
- Meminta kamu mengisi password
- Menanyakan info tambahan (boleh dikosongkan dengan menekan Enter)


Masukkan user ke dalam grup SUDO
=================================
> sudo usermod -aG sudo nama_user

-aG berarti "append to group"
Dengan ini, nama_user bisa menjalankan perintah dengan sudo


Verifikasi user ada di grup sudo
======================================
Login sebagai user baru, lalu jalankan:
> groups

Harusnya muncul output seperti ini:
nama_user sudo


Atau, kamu bisa cek dari user lain dengan:

> groups nama_user

===================
