INSTALL
=======
> sudo apt update && sudo apt upgrade -y
> sudo apt install nginx -y
> sudo systemctl status nginx
> sudo ufw allow 'Nginx Full'


Hardening Dasar Nginx
=====================
Sembunyikan versi Nginx
Edit file:
> sudo nano /etc/nginx/nginx.conf

Tambahkan:
server_tokens off;


Batasi request (anti DDoS ringan)
==================================
Tambahkan didalam : http {:

  server_tokens off;
  limit_req_zone $binary_remote_addr zone=one:10m rate=10r/s;
  server {
  
      limit_req zone=one burst=20 nodelay;
  
      add_header X-Frame-Options "SAMEORIGIN" always;
      add_header X-Content-Type-Options "nosniff" always;
      add_header Referrer-Policy "no-referrer-when-downgrade" always;
  
      add_header Content-Security-Policy "default-src 'self';" always;
      add_header Strict-Transport-Security "max-age=31536000; includeSubDomains" always;
  
  }


UPDATE NGINX
===============
ika menggunakan repository Ubuntu standar:
> sudo apt update
> sudo apt upgrade nginx


Atau untuk memastikan paket terbaru:
> sudo apt update
> sudo apt install --only-upgrade nginx


Cek versi sebelum dan sesudah update:
> nginx -v

Contoh output:
nginx version: nginx/1.24.0


Restart atau Reload Nginx
Setelah update:
> sudo systemctl restart nginx



Proteksi Tambahan Install Fail2Ban (anti brute force)
======================================================
a. Install Fail2Ban (anti brute force)
sudo apt install fail2ban -y

Enable:
> sudo systemctl enable fail2ban

b. Nonaktifkan service tidak perlu

Cek:
sudo systemctl list-unit-files --type=service



Buat konfigurasi Minimal local fail2ban
=======================================

> sudo nano /etc/fail2ban/jail.local

ISI MINIMAL :
[nginx-http-auth]
enabled = true
port = http,https
logpath = /var/log/nginx/error.log
maxretry = 5
bantime = 3600

ATAU AGAK LENGKAP :

[DEFAULT]
bantime = 30m
findtime = 10m
maxretry = 5
backend = systemd

# Optional (lebih ketat)
bantime.increment = true
bantime.factor = 2

[nginx-http-auth]
enabled = true
port = http,https
logpath = /var/log/nginx/error.log

[nginx-badbots]
enabled = true
port = http,https
logpath = /var/log/nginx/access.log

[nginx-req-limit]
enabled = true
port = http,https
logpath = /var/log/nginx/error.log
maxretry = 10


RESTART
> sudo systemctl restart fail2ban
